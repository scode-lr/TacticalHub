# TacticalHub - Multidive Manager

A comprehensive React TypeScript application for managing multiple diving activities, equipment, and dive planning.

## Features

### 🏊‍♂️ Dive Log Manager
- Track your diving history with detailed logs
- Record dive location, depth, duration, and environmental conditions
- Add notes and photos for each dive
- Monitor water and air temperatures, visibility

### 📋 Dive Planner
- Plan upcoming diving expeditions
- Select from predefined dive sites with difficulty ratings
- Coordinate with multiple divers
- Track weather conditions and tide information
- Manage emergency contacts

### 🛠️ Equipment Tracker
- Maintain inventory of diving equipment
- Track equipment types: tanks, regulators, BCDs, wetsuits, masks, fins, computers
- Monitor service schedules and maintenance dates
- Get notifications for equipment due for servicing

### 🗺️ Dive Sites Information
- Browse a comprehensive database of diving locations
- View detailed site information including:
  - GPS coordinates
  - Maximum depth
  - Difficulty level (Beginner to Expert)
  - Key features and marine life
- Add custom dive sites

## Technology Stack

- **React 19.1.1** - Modern React with hooks
- **TypeScript 4.9.5** - Type safety and better development experience
- **Create React App** - Standard React toolchain
- **CSS3** - Modern styling with gradients and animations
- **Responsive Design** - Mobile-friendly interface

## Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/scode-lr/TacticalHub.git
cd TacticalHub
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### Available Scripts

- `npm start` - Runs the app in development mode
- `npm test` - Launches the test runner
- `npm run build` - Builds the app for production
- `npm run eject` - Ejects from Create React App (one-way operation)

## Usage

### Navigation
The app features a intuitive tab-based navigation system:
- **Dive Logs**: Record and review your diving history
- **Dive Planner**: Plan future diving expeditions
- **Equipment**: Manage your diving gear and maintenance
- **Dive Sites**: Explore diving locations and add new sites

### Data Management
All data is stored locally in browser memory. For persistent storage, integrate with a backend database or local storage solution.
